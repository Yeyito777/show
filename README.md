# show

A tiny image and PDF viewer.

## Usage

```bash
show <file>
```

## Controls

- `Esc`: quit
- `Enter` / `f`: toggle fullscreen
- `=` / `+`: zoom in
- `-`: zoom out
- `0`: reset zoom and pan
- `h` / `j` / `k` / `l`: scroll left / down / up / right
- `Shift+j`: next PDF page
- `Shift+k`: previous PDF page
- `Home`: first PDF page

## Build

```bash
make
sudo make install
```
