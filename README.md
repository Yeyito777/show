# show

A tiny image and PDF viewer.

## Usage

```bash
show <file>
```

## Controls

- `Esc`: quit
- `Enter`: toggle fullscreen
- `Right` / `l` / `Space` / `PageDown`: next PDF page
- `Left` / `h` / `PageUp`: previous PDF page
- `0` / `Home`: first PDF page

## Build

```bash
make
sudo make install
```
